Clustering · Python

Customer Segmentation Analysis

Problem

~3,900 US shoppers looked identical on paper — one loyalty program, one email blast, no way to tell a bargain-hunter from a big spender.

Approach

Cleaned and scaled the data, used the elbow method to pick k, then ran K-Means clustering and profiled each group by income, age, and spending score.

Result

Five distinct customer segments emerged, each with a clear behavioral profile — a starting point for targeted campaigns instead of one-size-fits-all marketing.

3,900 shoppers → 5 clusters
EDA · Power BI

Istanbul Shopping Malls Analysis

Problem

Transaction data spanned 10 malls and 2021–2023, but leadership had no clear read on which customer group actually drove revenue.

Approach

Cleaned transaction and demographic data, modeled it into a star schema, and built a Power BI dashboard with slicers by mall, gender, category, and year.

Result

Found that 67.7% of total spend traced back to female customers — a clear signal for merchandising and where to point the next campaign.

10 malls · 2021–2023
Regression · Python

Startup Profit Analysis

Problem

50 startups across New York, California, and Florida — leadership wanted to know which spending category actually predicts profit.

Approach

Built a multiple linear regression model on R&D, admin, and marketing spend plus state, then checked it with backward elimination.

Result

Model reached R² = 0.95, with R&D spend emerging as the dominant driver of profit — the clearest lever for future investment decisions.
